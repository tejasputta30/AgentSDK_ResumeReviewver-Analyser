# 📄 AI Resume Analyzer & Reviewer using Agents SDK

## 📌 Overview
This project is an AI-powered Resume Analyzer and Reviewer built using an Agents SDK. It leverages intelligent agents to evaluate resumes, provide structured feedback, and suggest improvements based on industry standards.

## 🚀 Features
- Automated resume analysis
- AI-powered feedback and suggestions
- Multi-agent collaboration for evaluation
- Skill and content assessment
- Improvement recommendations

## 🛠️ Tech Stack
- Python
- Agents SDK
- OpenAI / LLM APIs


## ⚙️ How It Works
1. User uploads or inputs resume content
2. Multiple agents analyze different aspects:
   - Content quality
   - Skills and keywords
   - Formatting and structure
3. Agents collaborate to generate feedback
4. Final detailed review and suggestions are provided
